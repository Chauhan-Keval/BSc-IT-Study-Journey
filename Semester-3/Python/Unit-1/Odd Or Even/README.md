# Program: Odd Or Even

The program accepts input from the user and displays wheather it is odd or even.

- [Program Overview](#program-overview)
- [Output](#output)
- [How to Run](#how-to-run)
- [Dependencies](#dependencies)

## Program Overview

The Program asks for the value from the user and displays wheather given value is odd or even based on the condition like (val % 2 == 0) then even else odd value.

## Output

The program displays a sequence of numbers according to the following pattern:

```
    Enter a Number :
    Is Odd/Even/Zero Value
```

## How to Run

To run this program, follow the steps below:

1. Open a terminal.
2. Clone the repository:

   ```bash
   $ git clone https://github.com/Alpha-Hellboy/BSc-IT-Study-Journey
   ```

3. Navigate to the directory containing the program:

   ```bash
   $ cd semsester-n/language/program_folder
   ```

4. Compile the program using python interpretor.
5. Execute the compiled program:

   ```bash
   $ python3 app.py
   ```
